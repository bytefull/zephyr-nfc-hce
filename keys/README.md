# Keys generation

## 1. Generate P-256 private key

```bash
openssl ecparam -name prime256v1 -genkey -noout -out private_key.pem
```

## 2. Export public key PEM

```bash
openssl ec -in private_key.pem -pubout -out public_key.pem
```

## 3. Generate Zephyr-ready public key array

This extracts:
- raw uncompressed EC point
- exactly 65 bytes
- format: `0x04 || X || Y`

```bash
openssl ec -in private_key.pem -pubout -conv_form uncompressed -outform DER \
| tail -c 65 \
| od -An -tx1 -v
```

Zephyr code should look like this:

```c

	0x04, 0x89, 0x04, 0x9e, 0xbd, 0x0a, 0x86, 0xfd, 0x4d, 0xa3, 0x64, 0x82, 0x11,

```


## 4. Generate Android-ready private key array (PKCS#8 DER)

```bash
openssl pkcs8 -topk8 -nocrypt -in private_key.pem -outform DER -out private_key.der
```

Dump as hex:

```bash
od -An -tx1 -v private_key.der
```
